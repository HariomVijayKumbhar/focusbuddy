# FocusBuddy

FocusBuddy is a powerful focus and productivity application built with React, Vite, and Shadcn UI.

## Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your system.

### Installation

1.  **Navigate to the project directory:**

    ```bash
    cd focusbuddy-main
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

### Running the Project

To start the development server, run:

```bash
npm run dev
```

The application will be available at [http://localhost:3001](http://localhost:3001) (or the port specified in your terminal).

### Available Scripts

- `npm run dev` - Starts the development server.
- `npm run build` - Builds the project for production.
- `npm run lint` - Lints the project using ESLint.
- `npm run preview` - Previews the production build locally.
